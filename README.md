# Bird Radio

Static Web Audio player for a Moscow bird soundscape.

Open the GitHub Pages site, press `Start`, and keep the tab active for playback.

## Audio Notice

Bird recordings are sourced from Xeno-canto records listed in the app attribution panel. License, recordist, and source URL metadata are embedded in the page manifest for each clip. This public demo should be treated as a non-commercial listening prototype unless every included recording license is reviewed for the intended use.
